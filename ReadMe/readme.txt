Two ipython notebook file are included in the ReadMe directory.
GenSim_CeateStaticLibrary.ipynb: demonstrate how to create a static library for GenSim.
demo[1-3].ipynb: demonstrate several demos to use GenSim library and create a executable file.

1)If ipython notebook is installed, copy these two .ipynb files to the GenSim directory. Run the .ipynb file as

ipython notebook

2)In case ipython notebook is not installed, the .ipynb files have been converted to html files. Please copy those commands to your terminal to run it.
